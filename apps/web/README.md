# Vet Platform Web

Frontend em Next.js (App Router) para consumo da API do Vet Platform.

## Setup

1) Variavel de ambiente

Crie o arquivo `apps/web/.env.local` com:

```
NEXT_PUBLIC_API_URL=http://localhost:3001
```

2) Instalar dependencias

```
cd apps/web
npm install
```

3) Rodar

```
npm run dev
```

A aplicacao abre em http://localhost:3000.
